# Deep Learning with TensorFlow and PyTorch

## Overview

This repository contains my learning and practice work on deep learning using TensorFlow and PyTorch. It includes basic concepts, tensor operations, and simple neural network implementations.

The goal of this repository is to understand how deep learning frameworks work and how models are built step by step.

---

## Topics Covered

### TensorFlow

* Basics of TensorFlow
* Working with tensors
* Tensor operations
* Building simple neural networks

### PyTorch

* Tensor operations
* Autograd and gradients
* Neural network basics
* Model building and training
